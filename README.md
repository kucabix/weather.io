# weather.io
A React web application, that allows user to check the weather for the three available cities (Katowice, London, Sosnowiec). As the result, weather forecast is presented with more details for the current day, and less details for the next few days.

## Instalation

Clone the source repository from Github. On the command line enter:

```
git clone https://github.com/kucabix/weather.io.git
```

Project setup:

```
npm install
```

Project run:

```
npm start
```

## Built With

* [React](https://reactjs.org/) - (React version 16.8.6) The JS library
